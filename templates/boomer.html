    HELP MEE I'M A BOOMER!!!

    This website is for setting up combinations (profiles) of Maps, Gamemodes, and Mods to be played for Glorpnights. Granther can rotate the server to any profile

    Modpacks
    A Modpack is a container to hold many mods, a modpack can be used with many different profiles.
    Example Modpack:
    Mod: Low Gravity
    Mod: Grow taller after kill
    Mod: Fart noise on death

    Profiles
    Profiles are what represent a combination of a Map, Gamemode and Modpack. As of right now a Profile can only have ONE modpack.
    Example Profile:
    Gamemode: Micheal Myers
    Map: Sand
    Modpack: Glorp Modpack
